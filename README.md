# 🎓 Student Performance Dashboard (AIDS Project)

## 📌 Introduction
The Student Performance Dashboard is an interactive data analysis project developed under the Artificial Intelligence and Data Science (AIDS) domain. This project helps analyze student data like marks, attendance, and subject performance.

## 🎯 Objective
- Analyze student performance
- Identify trends and patterns
- Provide insights using visualization

## ⚙️ Technologies Used
- Python
- Pandas
- Plotly
- Streamlit

## 🚀 Features
- Upload CSV file
- Data analysis
- Interactive graphs
- Dashboard visualization

## ▶️ How to Run
pip install pandas streamlit plotly  
streamlit run app.py
